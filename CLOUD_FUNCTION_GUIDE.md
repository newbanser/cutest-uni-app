# 云函数部署指南

## 问题说明

小程序运行时出现 `FUNCTION_NOT_FOUND` 错误，是因为云函数尚未部署到微信云开发环境。

## 部署步骤

### 1. 开通云开发

1. 打开微信开发者工具
2. 点击顶部菜单「云开发」
3. 按照提示开通云开发（选择基础版即可）
4. 复制环境ID，确保与 `src/App.vue` 中的 `env` 一致（当前环境ID：`cloud1-d2gz1iuxn08055996`）

### 2. 部署云函数

项目中已包含以下云函数：

- `savePersonality` - 保存用户人格数据
- `getPersonality` - 获取用户人格数据
- `saveAnalysis` - 保存分析记录
- `createMatch` - 创建匹配记录
- `getMatches` - 获取匹配历史

部署步骤：

1. 在微信开发者工具左侧项目树中找到 `cloudfunctions` 文件夹
2. 右键点击每个云函数文件夹（如 `savePersonality`）
3. 选择「上传并部署：云端安装依赖」
4. 等待部署完成（状态变为绿色钩）
5. 重复以上步骤，部署所有云函数

### 3. 验证部署

部署完成后：

1. 在微信开发者工具点击「云开发」
2. 进入「云函数」标签页
3. 确认所有云函数都显示在列表中
4. 可以点击「测试」按钮测试云函数是否正常工作

## 本地测试（暂不使用云函数）

如果暂时不想部署云函数，小程序的核心功能（人格测试、关系匹配）仍然可以正常使用，因为所有数据都保存在本地存储中。云函数仅用于云端数据备份和好友匹配。
